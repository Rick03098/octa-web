# BirthdayInput 文件夹地图

## 地位（Role）
用户引导流程的第二个页面，负责收集用户的生日信息（年、月、日），使用轮盘选择器进行交互。

## 逻辑（Logic）
- **UI结构**: 
  - 生日渐变背景（绿色到粉色渐变）
  - 顶部左侧返回按钮
  - 居中标题"你的生日在\n哪一天"（白色，45px，ExtraBold）
  - 三个轮盘选择器容器（年、月、日）横向排列
  - 底部继续按钮（白色背景）
- **交互流程**: 
  1. 用户通过轮盘选择器选择年、月、日
  2. 用户点击"继续" → 验证日期有效性 → 保存到onboardingStore → 跳转到出生时间信息页
  3. 用户点击返回 → 跳转到姓名输入页
- **数据流**: 轮盘选择值 → onboardingStore.setBirthDate() → 状态更新 → 跳转到下一步

## 约束（Constraint）
- **严禁硬编码颜色**: 必须使用 `styles/variables.css` 中的CSS变量
- **背景渐变**: 使用生日页的渐变颜色
- **轮盘选择器**: 
  - 使用 `components/WheelPicker` 组件
  - 三个选择器：年、月、日
  - 选择器容器样式：渐变背景from #fffbf8 to #f7f3ef，边框rgba(239,136,58,0.2)，16px圆角，52px高度
- **标题样式**: 45px，白色，ExtraBold（800），letter-spacing: -0.45px，行高1.0
- **按钮样式**: 白色背景，58px高度，20px圆角，16px黑色文字
- **布局**: 三个轮盘选择器横向排列，宽度分别为：年136px，月57px，日63px

## 成员清单

- `BirthdayInputView.tsx` - **Birthday Input Gatekeeper**: 生日输入页面主组件，负责UI渲染、轮盘选择器交互和导航
- `BirthdayInputView.module.css` - **Style Processor**: 生日输入页面样式定义，使用CSS变量

## 轮盘选择器逻辑

### 年选择器
- 范围：1900年到当前年份
- 显示格式：`{年份}年`
- 默认值：当前年份 - 20（示例）

### 月选择器
- 范围：1到12
- 显示格式：`{月份}月`
- 默认值：当前月份

### 日选择器
- 范围：根据选择的年月动态计算（考虑闰年、月份天数）
- 显示格式：`{日期}日`
- 默认值：当前日期
