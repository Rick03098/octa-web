# /src/features/MainDashboardEmpty/.folder.md

## 文件夹地位 (POS)
特征层 (Feature Layer) 的首次打开主界面组件。它是用户第一次完成引导流程后进入的空白主界面，展示"添加你的第一个环境"界面，引导用户添加第一个环境。

## 逻辑职责 (Logic Responsibilities)
- **根据当前时间显示个性化问候语**：早上好/下午好/晚上好 + 用户名（Noto Serif SC Regular, 24px, #01031a，名字 #333333）
- **显示用户感受问题**："你今天的感受是怎样的？"（Noto Serif SC Regular, 16px, #333333）
- **显示渐变背景图**：位置 left=-9px, top=303px, width=400px, height=313.33px（从 Figma MCP 服务器获取）
- **显示空状态提示**：
  - 标题："添加你的第一个环境"（Source Han Serif SC Regular, 20px, white, 居中, top=348.5px）
  - 副标题："开启天人合一之境"（Source Han Serif SC Regular, 16px, rgba(255,255,255,0.97), 居中, top=403.5px）
- **显示搜索按钮**：GlassSearchButton 组件（left=169px, top=562px）
- **显示底部副标题**："开启天人合一之境"（Noto Serif SC Regular, 16px, rgba(255,255,255,0.97), 居中, top=892.5px）
- **处理导航操作**：
  - 搜索按钮 → 跳转到拍摄教程页面（/tutorial）

## 依赖关系 (Dependencies)
### 输入 (INPUT)
- `useOnboardingStore` (Zustand Store): 获取用户姓名 `name` 用于问候语
- `DSStrings` (Constants): 获取所有 UI 文本字符串
- `GlassSearchButton` (Component): 玻璃效果搜索按钮组件
- 渐变背景图：从 Figma MCP 服务器获取（http://localhost:3845/assets/1389-16380.png）
- CSS 变量：
  - `--color-bg-name-entry`: #fffcf9（背景色，与 NameEntry 页面相同）
  - `--color-text-primary`: rgb(26, 31, 51) / #01031a（问候语颜色）
  - `--color-text-tertiary`: #333333（名字和问题颜色）
  - `--color-text-white`: #ffffff（空状态标题颜色）
  - `--color-text-white-97`: rgba(255,255,255,0.97)（副标题颜色）
  - `--font-family-serif`: Noto Serif SC
  - `--font-family-source-han`: Source Han Serif SC
  - `--font-weight-regular`: 400
  - `--font-size-2xl`: 20px（空状态标题）
  - `--font-size-md`: 16px（问题、副标题）
  - `--font-size-4xl`: 24px（问候语）

### 输出 (OUTPUT)
- 导航至拍摄教程页面：`/tutorial`（点击搜索按钮）

## 约束 (Constraints)
- **布局约束**（Figma 精确像素值）：
  - 背景色：`#fffcf9`
  - 问候语：left=29px, top=109.5px (center-y), Noto Serif SC Regular, 24px, #01031a（名字 #333333）
  - 问题：left=29px, top=164.5px (center-y), Noto Serif SC Regular, 16px, #333333
  - 渐变背景图：left=-9px, top=303px, width=400px, height=313.33px
  - 空状态标题：left=197px (居中), top=348.5px (center-y), Source Han Serif SC Regular, 20px, white, width=290px, height=55px
  - 空状态副标题：left=197px (居中), top=403.5px (center-y), Source Han Serif SC Regular, 16px, rgba(255,255,255,0.97), width=290px, height=55px
  - 搜索按钮：left=169px, top=562px（使用 GlassSearchButton 组件，54x54px）
  - 底部副标题：left=197px (居中), top=892.5px (center-y), Noto Serif SC Regular, 16px, rgba(255,255,255,0.97), width=290px, height=55px
- **样式约束**：
  - 严禁硬编码色值，必须使用 CSS 变量
  - 所有位置和尺寸必须与 Figma 设计完全一致
  - 渐变背景图使用绝对定位，z-index=0
  - 所有文本元素使用绝对定位，z-index=1（文本在背景图上方）
  - 搜索按钮 z-index=10
- **交互约束**：
  - 搜索按钮可点击（跳转到拍摄教程页面）
  - 渐变背景图应添加 onError 处理，如果加载失败则隐藏

## 备注
- 该页面是"首次打开的主界面"设计，即用户完成引导流程后第一次进入主界面的空状态
- 背景使用纯色 #fffcf9（与 NameEntry 页面相同）
- 渐变背景图从 Figma MCP 服务器获取，需要容错处理
- 所有文本颜色、字体、尺寸必须严格按照 Figma 设计实现
- 该页面与 MainDashboard（有环境的主界面）是独立的两个页面，不应混淆
